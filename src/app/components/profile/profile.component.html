About The App

This app can be used to display Information about a user's github account

Guide:

  • Key in the user's account name in the search tool box on the left side
  • The profile picture with the username is displayed
  • Also, the number of:
    • Followers
    • Following
    • Public gists
    • Public repositories
  • A list of a user's Bio, whether available for hire and personal blog will also be displayed if the user has those details in his/her github account
